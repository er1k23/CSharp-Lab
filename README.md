# C# Lab

A collection of C# projects and exercises focused on language fundamentals, 
object-oriented programming, and applied problem solving.

## Contents

| Project | Description |
|---|---|
| [IEEE754-Float-Representation](./IEEE754-Float-Representation) | Exploring how floating-point numbers are represented in memory under the IEEE 754 standard |
| [Tic-Tac-Toe](./Tic-Tac-Toe) | Console-based Tic-Tac-Toe game built with OOP principles |

## Topics Covered

- Strings, collections, and string interning
- Recursion, enums with flags, nullable types, structs, tuples
- OOP: encapsulation, inheritance, polymorphism, abstract classes, interfaces
- Object relationships: IS-A, HAS-A, USES-A

## Tech Stack

C# / .NET
